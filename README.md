#### Напомню:
запуск в фоне:<br>
``` docker-compose up -d ```
 
запуск с пересборкой:<br>
``` docker-compose up -d --build ```

остановка<br>
``` docker-compose down ```

перезапуск с пересборкой:<br>
```docker-compose down && docker-compose up -d --build```
